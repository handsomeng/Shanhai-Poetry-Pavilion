## 🚨 常见被拒原因和解决方案

### 1. 隐私政策问题
**原因**：隐私政策 URL 无法访问或内容不完整

**解决**：
- 确保 GitHub URL 可以公开访问
- 检查 URL 是否正确（使用 raw.githubusercontent.com）
- 在浏览器中测试 URL

---

### 2. Sign in with Apple 配置问题
**原因**：Apple 登录未正确配置

**解决**：
- 在 Xcode 中确保添加了 "Sign in with Apple" Capability
- 在 Apple Developer 网站配置 App ID
- 确保 Provisioning Profile 包含该功能

---

### 3. 功能不完整或有 Bug
**原因**：审核人员发现崩溃或功能无法使用

**解决**：
- 提供详细的测试账号
- 在 "备注" 中说明测试步骤
- 确保所有功能都经过充分测试

---

### 4. 元数据问题
**原因**：描述、截图与实际功能不符

**解决**：
- 确保描述准确反映 App 功能
- 截图展示真实的 App 界面
- 不要夸大或误导

---

### 5. 内容审核问题
**原因**：用户生成内容未经审核

**解决**：
- 说明用户内容是私有的（仅用户自己可见）
- 如果有公开分享功能，需要内容审核机制
