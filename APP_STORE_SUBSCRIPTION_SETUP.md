## 📝 在 App Store Connect 中创建订阅

### 🎯 重要说明：免费试用在哪里设置？

**免费试用不在初始创建页面！** 设置流程：

```
1. 先创建订阅产品（只需填：类型、参考名称、产品ID）
   ↓
2. 创建后进入订阅详情页
   ↓
3. 在"订阅优惠"或"促销优惠"部分添加免费试用
   ↓
4. 选择"介绍性优惠" → "免费试用" → "7天"
```

现在开始创建：

---

### 第一步：创建订阅群组

1. 在 App Store Connect 中打开你的应用
2. 点击侧边栏的 **"订阅"**（Subscriptions）
3. 点击 **"创建订阅群组"**（Create Subscription Group）
4. 填写信息：
   - **参考名称**：`山海诗馆会员订阅`
   - **群组 ID**：系统自动生成（比如 `com.shanhai.poetry.subscriptions`）
5. 点击 **"创建"**

---

### 第二步：创建第一个订阅产品（月卡）

在刚创建的订阅群组中，点击 **"+"** 添加订阅：

#### 基本信息
- **参考名称**：`山海诗馆月卡会员`（仅内部使用，用户看不到）
- **产品 ID**：`com.shanhai.poetry.monthly`（⚠️ 必须与代码中一致！）

#### 订阅期限
- 选择：**1 个月**（1 Month）

#### 订阅价格
1. 点击 **"订阅价格"**（Subscription Prices）
2. 点击 **"添加本地化价格"**（Add Localized Price）
3. 选择地区：
   - **中国大陆**（China Mainland）
   - 价格：**¥9.9**（选择价格档位）
4. 其他国家可以选择"等值自动定价"或自己设置

#### 本地化信息（中国大陆）
点击 **"添加本地化"**，选择 **"简体中文"**：
- **订阅显示名称**：`月卡会员`（用户在 App Store 看到的名称）
- **描述**：`每月自动续费，畅享全部会员权益`

#### 免费试用（⚠️ 注意：这个选项在创建后才能看到）
**免费试用不在创建页面**，需要先完成基本创建，然后在详细配置页面中设置：

1. 创建订阅后，在订阅详情页
2. 找到 **"订阅优惠"**（Subscription Offers）或 **"促销优惠"** 部分
3. 点击 **"创建优惠"** 或 **"添加免费试用"**
4. 选择 **"介绍性优惠"**（Introductory Offer）
5. 类型选择：**"免费试用"**（Free Trial）
6. 时长：**7 天**
7. 保存

💡 **另一种方式**（更简单）：
- 在订阅定价页面，有些区域可以直接设置"免费试用期"
- 如果看不到，就在创建后通过"订阅优惠"添加

#### 审核信息
- **截图**：订阅页面的截图（展示订阅选项和权益）
- **审核备注**：说明订阅功能和权益

## ⚠️ 重要注意事项

### 1. 产品 ID 必须完全一致
代码中的产品 ID 必须与 App Store Connect 中的完全相同：
- `com.shanhai.poetry.monthly`
- `com.shanhai.poetry.quarterly`
- `com.shanhai.poetry.yearly`

### 2. 价格档位
App Store 有固定的价格档位，你需要选择最接近的：
- ¥9.9 → 选择 ¥12（如果没有 ¥9.9，选最接近的）
- ¥19.9 → 选择 ¥18 或 ¥25
- ¥69.9 → 选择 ¥68 或 ¥73

💡 **建议**：根据实际可用的价格档位调整代码中的显示价格。

### 3. 订阅群组
- 所有三个订阅必须在同一个订阅群组中
- 用户只能同时订阅一个产品
- 可以随时升级或降级

### 4. 审核材料
需要准备：
- 订阅页面截图
- 会员权益说明
- 自动续费和取消说明
- 价格和试用期说明

### 5. StoreKit 配置文件（本地测试）
在 Xcode 中：
1. 选择项目 → Capabilities → StoreKit Configuration
2. 创建 `.storekit` 配置文件
3. 添加三个订阅产品（使用相同的产品 ID）
4. 设置价格和试用期
5. 用于本地测试，无需真实购买
