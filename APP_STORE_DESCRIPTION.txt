【App 描述】（4000字符以内）

山海诗馆 - 你的诗意创作空间

【应用简介】
山海诗馆是一款专为现代诗爱好者打造的创作工具，让诗歌创作变得简单而优雅。无论你是诗歌初学者还是资深创作者，都能在这里找到属于自己的诗意表达方式。

【核心功能】

📚 学诗 - 系统化学习
• 6 大主题分类，20+ 篇原创教程
• 从基础入门到高级技巧的完整学习路径
• 现代诗语言、节奏、结构、情感表达全面讲解
• 经典诗歌赏析，提升审美品味

✍️ 写诗 - 多种创作模式
• 直接写诗：自由创作，实时字数统计
• 模仿写诗：参考经典，分屏对比学习
• 主题写诗：10 个意象主题，激发创作灵感
• AI 智能点评：从意象、情感、语言、结构四个维度分析你的作品

🌸 赏诗 - 欣赏优秀作品
• 浏览其他诗人的优秀创作
• 最新、热门、随机三种筛选方式
• 沉浸式阅读体验

👤 我的 - 作品管理
• 已发布、草稿、收藏分类管理
• 数据统计：作品数、草稿数、获赞数
• 完整的编辑和删除功能

【设计理念】

🎨 极简美学
• Dribbble 级别的高级设计
• 纯白背景，大量留白
• 极致轻盈的字体和排版
• 专注内容，回归诗歌本质

☁️ 数据安全
• iCloud 自动同步，多设备无缝切换
• 本地存储，保护隐私
• 支持数据导出和账号删除

🤖 AI 辅助
• 集成先进的 AI 技术
• 智能点评，提供专业建议
• 主题生成，激发创作灵感

【适合人群】

• 诗歌爱好者和创作者
• 想要学习现代诗写作的初学者
• 需要创作灵感和反馈的诗人
• 喜欢用文字记录生活的人

【为什么选择山海诗馆】

✓ 零基础友好：系统化教程，从入门到精通
✓ 创作自由：多种模式，满足不同创作需求
✓ AI 加持：智能点评，快速提升写作水平
✓ 优雅设计：极简美学，专注创作体验
✓ 数据安全：iCloud 同步，永不丢失

═══════════════════════════════════════════════

感谢审核！

【应用说明】
山海诗馆是一款诗歌创作工具，主要功能包括：
1. 学习现代诗创作技巧
2. 多种模式写诗（直接写、模仿写、主题写）
3. AI 智能点评
4. 作品管理和分享

【测试建议】
1. 使用 Apple ID 登录（或跳过登录使用游客模式）
2. 在"学诗"页面浏览教程
3. 在"写诗"页面尝试创作一首诗
4. 点击"AI 点评"查看智能反馈（需要 API Key）
5. 在"我的"页面查看作品管理功能

【技术说明】
- 使用 Sign in with Apple 进行身份验证
- 支持游客模式（跳过登录）
- 数据通过 iCloud Key-Value Storage 同步
- AI 功能使用 DeepSeek API（用户需自行配置 API Key）

【重要提示】
- 游客模式下所有功能均可使用，数据仅存储在本地
- 登录后数据会自动同步到 iCloud
- AI 点评功能需要用户在设置中配置自己的 API Key
